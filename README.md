### About The Project
A modern, responsive personal portfolio website designed with a "Minimal Paper" aesthetic. Built to showcase my projects, skills, and experience as an Electronics Computer Technology student. The design focuses on readability, clean lines, and a professional yet approachable feel.

### Built With
-   [React](https://reactjs.org/) - A JavaScript library for building user interfaces.
-   [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for rapid UI development.
-   [Vite](https://vitejs.dev/) - Next Generation Frontend Tooling.
-   [EmailJS](https://www.emailjs.com/) - Send email directly from JavaScript (no server required).
-   **Framer Motion / Headless UI** - For smooth animations and accessible components.

### Features
-   **Minimal Paper Theme**: Clean, high-contrast design with paper-like cards and earth tones.
-   **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
-   **Project Showcase**: Interactive cards displaying my work with images, descriptions, and links to GitHub/Live Demos.
-   **Contact Form**: Functional email form integrated with EmailJS for direct communication.
-   **Experience Timeline**: A structured view of my educational and professional background.
-   **Asset Optimization**: Optimized image loading and PDF viewing support for resumes/transcripts.

### Getting Started
1.  **Clone the repo**:
    ```sh
    git clone https://github.com/Tanongsak-BKK/my-portfolio.git
    ```
2.  **Install dependencies**:
    ```sh
    npm install
    ```
3.  **Run locally**:
    ```sh
    npm run dev
    ```
4.  **Build for production**:
    ```sh
    npm run build
    ```
